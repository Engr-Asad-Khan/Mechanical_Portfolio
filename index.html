CAD Projects

SolidWorks: Single Plate Clutch Assembly – 3D Model and Exploded View

This project showcases a fully modeled single plate clutch assembly, created using SolidWorks. The clutch is a key component in automotive power transmission systems, enabling smooth engagement and disengagement between the engine and the transmission.

The model includes all essential parts:

  • Clutch Plate (Friction Disc)
  • Pressure Plate
  • Flywheel
  • Diaphragm Spring
  • Clutch Cover
  • Release Bearing
  • Fasteners and Housing

The exploded view highlights the assembly hierarchy and relationship between components, demonstrating a clear understanding of mechanical design principles and spatial arrangement.

Key Features:

  • Designed with precise mating conditions and assembly constraints
  • Realistic material selection and surface appearances for visual clarity
  • Detailed exploded view for presentation and manufacturing reference
  • Created with manufacturability and functionality in mind

This project demonstrates proficiency in SolidWorks 3D modeling, assembly design, exploded view generation, and technical presentation for mechanical components.
